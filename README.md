üöÄ Project Steps
1. Data Ingestion & EDA (Python)
Data preparation and cleaning were performed in Python.
Used pandas for data loading and initial exploration with df.info() and .describe().
Handled missing data in the Review Rating column by using the median rating of each product category for imputation.
Renamed columns to snake case and ensured data consistency.
Feature Engineering: Created age_group and purchase_frequency_days columns.
2. SQL Analysis (PostgreSQL)
The cleaned DataFrame was loaded into PostgreSQL for structured analysis. Key business questions answered included:
Comparing total revenue generated by gender.
Identifying high-spending customers who used discounts.
Comparing average purchase amounts between standard and express shipping types.
Analyzing average spend and total revenue across subscription statuses.
Segmenting customers into New, Returning, and Loyal groups based on purchase history.
3. Power BI Dashboard
An interactive dashboard was built in Power BI to present insights visually. Key metrics displayed included the number of customers (3.9K), average purchase amount ($59.76), and average review rating (3.75).
4. Final Reporting
A detailed report was created, and findings were communicated through a presentation generated via Gamma.
